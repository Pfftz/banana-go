# Banana Go (Placeholder Mockup)

Proyek ini saat ini adalah mockup sederhana struktur halaman untuk e-commerce camilan pisang "Banana Go". Semua halaman masih placeholder dan belum ada fungsionalitas dinamis.

## Halaman Saat Ini

| Route               | Deskripsi                                |
| ------------------- | ---------------------------------------- |
| `/`                 | Halaman utama dengan hero & contoh rasa. |
| `/katalog`          | Daftar placeholder rasa produk.          |
| `/pemesanan`        | Keranjang & checkout dummy.              |
| `/kontak-testimoni` | Form kontak & testimoni awal.            |

## Tujuan Mockup

-   Menyediakan kerangka desain awal bagi tim UI/UX.
-   Memvalidasi struktur navigasi dan copy awal.
-   Memudahkan pembagian tugas pengembangan fitur selanjutnya (cart, auth, dsb).

## Langkah Berikutnya (Rencana)

1. Tambah data produk nyata + gambar.
2. Implementasi state keranjang belanja.
3. Integrasi form kontak & testimoni backend.
4. Optimasi SEO & performa.
